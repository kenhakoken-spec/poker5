# ⚡ Quick Start Guide - 爆速入力システム

5分で理解！ポーカーハンドトラッカーの使い方

---

## 🎮 基本操作（3ステップ）

### ステップ1: ポジション選択

アプリ起動時に表示される画面で、あなたのポジションを選択：

```
┌─────────────────────────────────┐
│  あなたのポジションを選択       │
│                                 │
│  [SB] [BB] [UTG]                │
│  [HJ] [CO] [BTN]                │
└─────────────────────────────────┘
```

例: BTN を選択

---

### ステップ2: アクションを記録

#### プリフロップ（爆速！）

```
┌─────────────────────────────────┐
│  アクションするプレイヤーを選択 │
│  （前のプレイヤーは自動Fold）   │
│                                 │
│  [UTG] [HJ] [CO]                │
│  [BTN] [SB] [BB]                │
└─────────────────────────────────┘
```

**例: COがレイズ**
1. CO をタップ
2. Raise をタップ
3. 3x (4.5bb) をタップ

→ UTG, HJ は自動的にFold！

**例: BTNが直接レイズ**
1. BTN をタップ
2. Raise をタップ
3. 2x (3.0bb) をタップ

→ UTG, HJ, CO は自動的にFold！

#### ポストフロップ

```
┌─────────────────────────────────┐
│  現在の手番: BB                 │
│  （他のプレイヤーは無効）       │
│                                 │
│  [BB] ✓ 選択可能                │
│  [BTN] × 無効                   │
└─────────────────────────────────┘
```

1. 現在の手番のプレイヤーを選択
2. アクション（Check/Bet/Fold/Call）を選択
3. ベットの場合、サイズを選択

---

### ステップ3: ボードを入力

```
┌─────────────────────────────────┐
│  [Board] ボタンをタップ         │
│                                 │
│  Flop: A♠ K♥ Q♦                │
│  Turn: J♣                       │
│  River: 10♠                     │
└─────────────────────────────────┘
```

各ストリートのカードを選択すると、自動的に次のストリートへ遷移！

## 🚀 爆速入力の例

### シナリオ: COオープン、BTN 3-bet、BBコール

#### 従来の方法（7ステップ）
```
1. UTG → Fold
2. HJ → Fold
3. CO → Raise 3bb
4. BTN → Raise 9bb
5. SB → Fold
6. BB → Call
7. [次のストリート]
```

#### 新しい方法（4ステップ）
```
1. CO → Raise 3bb      (UTG, HJ 自動Fold)
2. BTN → Raise 9bb     (自動で進む)
3. SB → Fold           (自動で進む)
4. BB → Call           (自動で進む)
5. [自動遷移]
```

**43%の時間短縮！**

## 💡 プリセットボタン活用法

### プリフロップ

```
┌─────────────────────────────────┐
│  [2x]  → ポットの2倍            │
│  [3x]  → ポットの3倍            │
│  [Custom] → 任意のサイズ        │
└─────────────────────────────────┘
```

**使い方:**
- スタンダードオープン: 3x
- 3-bet: 3x (相手のレイズの3倍)
- 4-bet: 2x (相手の3-betの2倍)

### ポストフロップ

```
┌─────────────────────────────────┐
│  [33%]  → ポットの1/3           │
│  [50%]  → ポットの半分          │
│  [75%]  → ポットの3/4           │
│  [100%] → ポットベット          │
│  [150%] → オーバーベット        │
│  [All-in] → 全額                │
│  [Custom] → 任意のサイズ        │
└─────────────────────────────────┘
```

**使い方:**
- Cベット: 33%～50%
- バリューベット: 75%～100%
- ブラフ: 50%～75%
- オーバーベット: 150%

## 🎨 UI要素の意味

### ポジションボタンの色

```
🔵 青色 + リング = 現在の手番
🟡 黄色 = あなた（ヒーロー）
⚪ グレー = アクティブプレイヤー
⚫ 暗いグレー = フォールド済み
```

### アクションボタンの色

```
🔴 赤色 = Fold
🔵 青色 = Check / Call
🟡 黄色 = Bet / Raise (プリセット)
🟠 オレンジ = オーバーベット
🟣 紫色 = All-in
⚪ グレー = Custom
```

## 🐛 トラブルシューティング

### Q: ポストフロップで手番が進まない
**A:** 全員のアクションが完了しているか確認してください。全員がCheckまたはCallした時点で次のストリートへ自動遷移します。

### Q: プリフロップでスキップ機能が動かない
**A:** 新版（エンジン搭載）を使用していることを確認してください。右上のボタンで新版に切り替えられます。

### Q: ボタンが押せない
**A:** 現在の手番ではないポジションは無効化されています。青色でハイライトされているポジションのみ選択可能です。

### Q: ベットサイズがおかしい
**A:** ポストフロップのパーセンテージは「ポット」に対する割合です。例えば50%は「現在のポットの半分」という意味です。
